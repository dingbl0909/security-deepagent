# 私有化部署与接口异常排查 SOP

## 适用场景

安防平台在本地工作站、边缘服务器或私有化环境部署后出现接口异常、服务启动失败、检索为空或告警无法写入。

## 排查顺序

1. 检查 `.env`、配置文件和服务端口是否与部署环境一致。
2. 检查 SQLite、日志目录、知识库目录和工作区目录是否存在并具备写权限。
3. 检查模型服务地址、API Key 和 OpenAI 兼容接口是否可访问。
4. 检查 FastAPI 服务健康检查 `/health` 是否返回 `ok`。
5. 检查业务接口 `/chat`、线程查询 `/threads/{thread_id}` 和人工确认 `/review/continue`。

## 常见原因

- 未设置 `PYTHONPATH=src` 或未安装项目包，导致 `security_agent` 无法导入。
- 数据库未初始化，缺少设备和告警样例数据。
- 知识库目录为空，导致无法返回证据。
- LLM 开关打开但没有配置 `OPENAI_API_KEY`。
- 本地模型服务未启动或 base url 配置错误。

## 风险动作

修改生产配置、删除数据库、清空日志、重启模型服务或重启平台服务前需要人工确认。
