# 08 前端工作台设计

源头文档：`项目介绍.txt`

溯源段落：
- 前端采用 React + TypeScript + Vite 构建安防智能工作台。
- 提供智能对话、证据展示、ReAct 轨迹、任务清单、人工确认、设备/告警概览、图片上传研判等能力。

## 3. 高风险展示

当 `/chat` 返回 `interrupted=true`：

- 展示风险原因。
- 展示建议动作。
- 展示批准/拒绝按钮。
- 审批后触发 resume，并展示恢复后的回答。

---

## 4. 验收

- 首屏是可用工作台。
- answer/evidence/trace/tasks/review 都能展示。
- 图片上传有大小和类型校验。
- 高风险中断状态清晰可见。
