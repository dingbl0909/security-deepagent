# 09 部署与配置设计

源头文档：`项目介绍.txt`

溯源段落：
- 项目面向私有化部署。
- 后端 FastAPI，前端 React + Vite，运行时依赖 DeepAgents、存储和模型服务。

覆盖任务：
- T11 配置管理
- T13 `/ready`

---

## 1. 中间件

`deploy/` 保留 PostgreSQL、Redis、Milvus、MinIO 等中间件编排。

---

## 2. 环境变量

必需：

- `DATABASE_URL`
- `REDIS_URL`
- `MILVUS_URI`
- `OPENAI_BASE_URL`
- `OPENAI_API_KEY`
- `SECURITY_AGENT_MODEL`

多模型：

- `PRIVATE_LLM_API_KEY`
- `PUBLIC_LLM_API_KEY`
- `PUBLIC_VISION_API_KEY`

---

## 3. Readiness

`/ready` 检查：

- PostgreSQL
- Redis
- Milvus
- LLM Gateway
- Sub-agent 配置
- model_profile 配置

---

## 4. 验收

- 依赖不可用时 `/ready` 返回失败。
- secret 不写入日志。
- 公开 API 配置必须声明数据策略。
