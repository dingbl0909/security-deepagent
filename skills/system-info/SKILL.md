# 系统信息采集 Skill

## 能力边界

用于本地工作站部署排障时收集环境信息和检查运行条件。

## 可用工具

- 首版不开放直接读取宿主机任意文件或执行 shell。
- 可结合用户提供的错误信息、配置路径和系统提示进行排障。
- 如需确认服务、数据库或知识库状态，应优先说明需要用户补充的命令输出或日志片段。

## 检查项

- Python 版本和虚拟环境。
- FastAPI 服务端口。
- SQLite 数据库路径和写权限。
- 知识库目录是否存在。
- `.env` 中模型服务地址和 API Key 是否配置。

## 安全要求

不要读取项目目录外的敏感文件。涉及删除、重启、修改系统配置的操作必须先请求人工确认。
